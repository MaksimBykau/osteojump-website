# Контент-план: Беременность, роды, дети (2026-03)

## Структура статей

1. **Пилар-статья "Остеопатия при беременности"** (2000-3000 слов)
   - Токсикоз, отёки, боли, тревожность, слабость
   - Положение плода, частота сеансов
2. **"Восстановление после родов"** (1200-1800 слов)
   - Диастаз, опущение органов, кесарево/ест. роды, рубцы
3. ~~"Остеопатия для новорождённых"~~ — **ГОТОВО** (slug: `osteopatia-dla-noworodkow-i-niemowlat`)
   - Кривошея, тонус, выравнивание черепа
4. **"Подготовка к беременности"** (800-1200 слов)
   - Зачатие, планирование
5. **FAQ — типы родов, частота сеансов** (3-4 вопроса)

## Дополнительные темы (анализ)

- [ ] Грудное вскармливание — боли у мамы, захват у малыша. Очень частый запрос.
- [ ] На каких сроках можно/нельзя — первый вопрос любой беременной. Без ответа не запишется.
- [ ] Мифы и страхи — "а вдруг навредит ребёнку?" Развенчание = конверсия.
- [ ] Тазовое предлежание — отдельный акцент, высокочастотный запрос.
- [ ] Противопоказания — честность повышает доверие.

## Нюансы

- Темы о типах родов (роддом/вода/дом/вертикальные) — деликатно. Лучше как FAQ в мягкой формулировке ("мой опыт акушерки показал..."), а не отдельные статьи.
- "Эстетика" — уточнить: лицо или тело? От этого зависит куда встроить.
- Тревожность — сильный дифференциатор, у Ханны есть сертификат по сомато-эмоциональному релизу. Подчеркнуть.
