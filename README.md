# OsteoJump

Информационный сайт об остеопатии, помогающий людям понять когда им необходимо обратиться к остеопату.

## Особенности

- Поддержка 5 языков: русский, английский, немецкий, польский, украинский
- Полностью адаптивная верстка для всех устройств
- Автоматический деплой на GitHub Pages при коммите в main ветку
- Быстрая загрузка без зависимостей
- Современный дизайн с плавными переходами
